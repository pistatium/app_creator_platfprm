API利用上の注意

  • APIは自己責任にてご利用下さい。
  • APIは予告なく仕様変更、提供中止する場合がございます。
  • キャッシュにより最新のデータが取得できない場合がございます。
  • APIを経由しないデータの利用(スクレイピングなど)はご遠慮下さい。
  • 過度にサーバーに負荷を掛けるような利用方法はご遠慮下さい。
  • 大量のリクエストが予想される場合は事前にご連絡下さい。
  • 放課後アプリ部へのリンクを貼ってもらえると助かります。

APIで何ができるのか

  • プラットフォーム別アプリ一覧取得
  • 開発者別アプリ一覧
  • アプリ詳細取得

API利用方法

共通パラメーター

HTTP Method: GET

API HOST: http://houkago-no.appspot.com

client
(必須)クライアントID、API申請画面から取得
key
(必須)クライアントIDと対になるAPIキー、API申請画面から取得
version
(必須)「1」を入れて下さい。

プラットフォーム別アプリ一覧取得API

概要

プラットフォーム別の登録アプリが最新順で取得できます。

URL

{% for p in platforms %}
{{ p.1 }}アプリ一覧
{% url app.views.api.recent_app p.1 %}
{% endfor %}

拡張パラメータ

count
(任意)結果取得件数、デフォルト:10、最大:50
offset
(任意)結果のオフセット、デフォルト:0

開発者別アプリ一覧API

概要

開発者ごとのプラットフォーム別登録アプリが最新順で取得できます。自分の開発したアプリを取得するのに便利です。

URL

{% for p in platforms %}
{{ p.1 }}アプリ一覧
{% url app.views.api.developer_apps developer.user_alias|default:"developer_id" p.1 %}
{% endfor %} (※取得したいユーザーに応じてdeveloper_idを変えてください。)

拡張パラメータ

count
(任意)結果取得件数、デフォルト:10、最大:50
offset
(任意)結果のオフセット、デフォルト:0

アプリ詳細取得API

概要

一覧で取得したIDをキーにアプリの詳細が取得できます。

URL

{% url app.views.api.detail_app 12345 %} (※12345はアプリID)

拡張パラメータ

なし
